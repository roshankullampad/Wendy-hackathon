# Wendy's AI Agents Hackathon Agenda

## Objectives
- Build a multi-agent workflow that discovers market trends, analyzes customers,
  and prepares a 2026 event calendar.
- Synthesize findings into 3 offer concepts for Wendy's.

## Deliverables
1. Modular agent architecture using Google ADK patterns.
2. Streamlit UI to run the workflow and display results.
3. Structured outputs for trend briefs, customer insights, events, and offers.

## Suggested Flow
1. Market Trends Analyst
2. Customer Insights Manager
3. Event Planner
4. Offer Orchestrator and Offer Design

## Demo Script
1. Open the Streamlit UI.
2. Enter a research query (for example: "Gen Z late night craving trends").
3. Run the workflow in the Query tab.
4. Review offer concepts in the Results tab.
